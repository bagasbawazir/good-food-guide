## Quick Start

1.  Clone the project

2.  Navigate to your new project:

```bash
$ cd good-food-guide
```

3.  Install all dependencies:

```bash
$ yarn
# or
$ npm install
```

4.  Start the dev server and edit some code!

```bash
$ yarn dev 
# or
$ npm run dev
```
